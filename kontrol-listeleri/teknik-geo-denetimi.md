# Teknik GEO Denetimi Kontrol Listesi

Bu kontrol listesi, bir web sitesinin yapay zeka cevap motorları için teknik olarak ne kadar hazır olduğunu değerlendirmek için hazırlanmıştır.

GEO burada **Generative Engine Optimization** anlamında kullanılır.

## 1. AI bot erişimi

- [ ] robots.txt dosyası erişilebilir mi?
- [ ] Googlebot engellenmiyor mu?
- [ ] Bilinen yapay zeka crawler’ları tamamen engellenmiyor mu?
- [ ] Sayfada noindex etiketi yok mu?
- [ ] Sayfada nofollow direktifi gereksiz şekilde kullanılmıyor mu?
- [ ] nosnippet veya çok kısıtlayıcı max-snippet direktifi yok mu?

## 2. llms.txt kontrolü

- [ ] /llms.txt dosyası var mı?
- [ ] llms.txt içinde site ve marka kısa şekilde tanıtılıyor mu?
- [ ] Önemli sayfalara bağlantılar var mı?
- [ ] Ürün, kategori veya hizmet açıklamaları net mi?
- [ ] İçerik yapay zeka sistemleri için kısa ve anlaşılır mı?

## 3. Sitemap ve keşfedilebilirlik

- [ ] XML sitemap erişilebilir mi?
- [ ] Sitemap içinde önemli sayfalar var mı?
- [ ] robots.txt içinde sitemap yolu belirtilmiş mi?
- [ ] Önemli sayfalar dahili linklerle destekleniyor mu?

## 4. Schema ve yapılandırılmış veri

- [ ] Organization schema var mı?
- [ ] WebSite schema var mı?
- [ ] WebApplication, Product, Article veya FAQ schema gerekiyorsa kullanılmış mı?
- [ ] JSON-LD hatasız mı?
- [ ] Marka adı, logo ve URL tutarlı mı?

## 5. Sayfa içi teknik sinyaller

- [ ] Title etiketi net mi?
- [ ] Meta description açıklayıcı mı?
- [ ] Sayfada tek ve anlamlı H1 var mı?
- [ ] H2 ve H3 başlıkları düzenli mi?
- [ ] Canonical etiketi doğru mu?
- [ ] hreflang gerekiyorsa doğru kullanılmış mı?

## 6. Makine-okunabilir içerik

- [ ] Sayfa içeriği JavaScript olmadan temel olarak okunabiliyor mu?
- [ ] Ana içerik HTML içinde net şekilde yer alıyor mu?
- [ ] Gereksiz popup veya engelleyici katmanlar yok mu?
- [ ] İçerik kısa, net ve konu odaklı mı?
